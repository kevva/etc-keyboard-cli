# etc-keyboard-cli [![Build Status](https://travis-ci.org/kevva/etc-keyboard-cli.svg?branch=master)](https://travis-ci.org/kevva/etc-keyboard-cli)

> Detect keyboard layout on Linux


## Install

```
$ npm install --global etc-keyboard-cli
```


## Usage

```
$ etc-keyboard --help

  Usage
    $ etc-keyboard
    us
```
